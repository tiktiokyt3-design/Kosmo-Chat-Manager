# Kosmo-Chat-Manager 🔥

📨 Kosmo | Чат-менеджер. Хорошой бот, отлично подходит к небальшой беседе, группе.

⭐ В боте присутствует такие команды: /start (private chat only), /profile, /add_role, /ban, /kick, /unmute, /unban, /pin, /unpin (only public chat).

📝 Так-же еще будут выходить обновления в боте будут больше команд новые системы, модули.


✅ Еще есть обработчик команд без /: профиль, панель, бан, кик, унбан, унмут, мут, пин, унпин.

‼️ Панель - это система админ панель одним словом. Позволяет владельцу боту управлять ботом в нутри телеграма.

🔥 Система ролей - роли: user, moderator, admin, st.admin, owner, manager, dayn, owner_bot.

🆙 Иархия ролей:

    "user": 0,

    "admin": 2,

    "owner": 4,
